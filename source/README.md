	------ But du Projet ------
Projet de recherche et de programmation dans le cadre d'un stage.

	------ Aide ------
Une aide est a disposition dans le répertoire "Aide" ou via le programme dans le menu onglet Aide


	------ Fonctionnalité ------
- Génération de grille aléatoire (avec option "résolvable")
- Génération de grille avec les règle du jeu Robot ricochet
- Editeur de grille
- Sauvegarde / charger
- Possibilité de lire un fichier contenant une liste de coup
- Resolveur adapter pour les deux type de grille (aléatoire ou Robot ricochet)
